# MAT-Network-Scanner-
Its an Security Tool which is used to scan the network and find services running on them built on python programming langauge 
It is easy yet effective to do find services or scan network 

This will be a Python-based CLI/GUI network scanner that:

* Discovers hosts on a network (ping sweep)
* Scans open ports on discovered hosts
* Identifies services running on those ports (banner grabbing)
* Displays results in a clean terminal UI

NOTE - please pip install all neccessary files
